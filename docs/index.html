How to install

  1. Download the extension zip file
  2. Unzip the file into a directory you will remeber
  3. Go to chrome://extensions/
  4. Hit the developer mode toggle
  5. Hit "Load unpacked"
  6. Select the folder in which you unziped the extension
  7. You should now see "Revisionist History" in your extensions!
  8. If it disapears due to chrome updates, etc. Repeat this proccess.

Features

  • Complete Writing Timeline: Track editing sessions, time spent writing, and revision patterns
  • Unusual Writing Pattern Detection: Identify atypical writing patterns that may benefit from additional guidance (these patterns are NOT evidence of any specific behavior and may have perfectly reasonable explanations)
